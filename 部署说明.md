# 云服务器部署说明

## 系统要求

- Linux（Ubuntu 20.04 / 22.04 推荐）
- Node.js 18+
- 至少 1GB 内存（Puppeteer/Chromium 比较吃内存）

---

## 第一步：安装系统依赖

Chromium 在 Linux 上需要一些系统库，运行以下命令一次性安装：

```bash
sudo apt-get update
sudo apt-get install -y \
  ca-certificates fonts-liberation \
  libappindicator3-1 libasound2 libatk-bridge2.0-0 libatk1.0-0 \
  libc6 libcairo2 libcups2 libdbus-1-3 libexpat1 libfontconfig1 \
  libgbm1 libgcc1 libglib2.0-0 libgtk-3-0 libnspr4 libnss3 \
  libpango-1.0-0 libpangocairo-1.0-0 libstdc++6 libx11-6 \
  libx11-xcb1 libxcb1 libxcomposite1 libxcursor1 libxdamage1 \
  libxext6 libxfixes3 libxi6 libxrandr2 libxrender1 libxss1 \
  libxtst6 lsb-release wget xdg-utils
```

---

## 第二步：安装 Node.js（如未安装）

```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs
```

---

## 第三步：上传项目文件

将 `cloud/` 文件夹内所有文件上传到服务器，例如上传到 `/home/ubuntu/whatsapp-bot/`。

```
whatsapp-bot/
├── src/
├── public/
├── config/
├── package.json
├── ecosystem.config.js
└── .env.example
```

---

## 第四步：安装依赖 & 配置环境变量

```bash
cd /home/ubuntu/whatsapp-bot
npm install

# 复制并编辑环境变量
cp .env.example .env
nano .env
```

在 `.env` 中填写密码（**强烈建议**）：
```
PORT=3000
AUTH_USER=admin
AUTH_PASS=你的密码
```

---

## 第五步：安装 PM2 并启动

```bash
# 安装 PM2（全局）
sudo npm install -g pm2

# 启动机器人
pm2 start ecosystem.config.js

# 设置开机自启
pm2 startup
pm2 save
```

常用 PM2 命令：
```bash
pm2 logs whatsapp-bot      # 查看实时日志
pm2 status                 # 查看运行状态
pm2 restart whatsapp-bot   # 重启
pm2 stop whatsapp-bot      # 停止
```

---

## 第六步：扫码登录

1. 确保服务器防火墙开放 3000 端口（或你配置的端口）
2. 浏览器打开 `http://服务器IP:3000`
3. 输入用户名密码登录
4. 扫描页面上的 QR 码完成 WhatsApp 登录
5. 登录成功后 QR 码消失，显示任务列表

---

## 第七步：配置任务

在管理界面新增定时任务，填写群组名称、消息内容和发送时间。

---

## 防火墙设置（Ubuntu UFW）

```bash
sudo ufw allow 3000/tcp
sudo ufw enable
```

> 建议配合 Nginx 反向代理 + HTTPS，避免密码明文传输。

---

## 注意事项

- `data/` 目录保存 WhatsApp 登录会话，**不要删除**，否则需要重新扫码
- 服务器重启后 PM2 会自动重启机器人（需先执行 `pm2 startup && pm2 save`）
- 如果消息发送失败并看到 "detached Frame" 错误，程序会自动重连，无需手动干预
